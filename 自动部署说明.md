# 🚀 一键自动部署脚本使用说明

## 使用方法

**双击运行** `自动部署.bat`，脚本会自动完成：

1. ✅ 检查Git是否安装
2. ✅ 配置Git用户信息
3. ✅ 初始化Git仓库
4. ✅ 添加所有文件
5. ✅ 提交代码
6. ✅ 配置远程仓库
7. ✅ 推送到GitHub

## 如果推送时提示认证

脚本会在推送前暂停，提示你准备Token。

### 获取Personal Access Token：

1. **访问**: https://github.com/settings/tokens
2. **点击**: "Generate new token (classic)"
3. **填写**:
   - Note: `Git Push`
   - 勾选 `repo` 权限（必须！）
4. **生成并复制Token**（只显示一次！）
5. **推送时**:
   - 用户名: `duongthitam57648-svg`
   - 密码: **粘贴Token**（不是GitHub密码）

## 推送成功后

1. 访问 https://github.com/duongthitam57648-svg/EA-
2. 点击 **Settings** → **Pages**
3. Source 选择：**GitHub Actions** ✅
4. 等待2-3分钟自动部署完成
5. 访问：https://duongthitam57648-svg.github.io/EA-/

## 如果推送失败

脚本会显示错误信息和解决方案，按照提示操作即可。

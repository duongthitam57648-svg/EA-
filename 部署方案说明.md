## 🎯 推荐方案：Vercel部署

### 为什么选择Vercel？

1. **完全免费** - 个人项目完全够用
2. **5分钟部署** - 最简单快捷
3. **全球CDN** - 访问速度快
4. **自动HTTPS** - 安全可靠
5. **手机直接访问** - 无需配置

### 快速开始

1. **准备GitHub仓库**（如果没有）
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/your-username/your-repo.git
   git push -u origin main
   ```

2. **部署到Vercel**
   - 访问 https://vercel.com
   - 登录GitHub账号
   - 点击 "New Project"
   - 选择你的仓库
   - 点击 "Deploy"

3. **完成！**
   - 几分钟后获得链接：`https://your-project.vercel.app`
   - 这个链接可以在任何设备上访问

## 📱 使用部署后的链接

部署完成后：

1. 打开"接收数据"对话框
2. 生成的链接会自动使用Vercel域名
3. 或者手动输入Vercel域名
4. 生成的链接格式：`https://your-project.vercel.app/?data=...`
5. 手机直接打开即可！

## 🔄 更新部署

每次修改代码后：
- 推送到GitHub
- Vercel自动重新部署
- 链接保持不变，内容自动更新

## 💡 其他方案

### 如果不想部署，使用内网穿透：

```bash
# 安装localtunnel
npm install -g localtunnel

# 启动开发服务器
npm run dev

# 在另一个终端运行
lt --port 3000
```

会生成一个临时链接，例如：`https://xxx.loca.lt`

**注意**：免费版链接会变化，每次重启都会生成新链接。

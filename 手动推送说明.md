# 🚀 手动推送步骤

由于PowerShell环境限制，请按照以下步骤手动推送：

## 方法1：使用批处理文件（推荐）

1. **双击运行** `推送.bat` 文件
2. 如果提示输入用户名和密码：
   - 用户名：`GHVBUGB`
   - 密码：使用Personal Access Token（不是GitHub密码）

## 方法2：手动执行命令

打开 **CMD**（不是PowerShell），执行以下命令：

```cmd
cd /d "C:\Users\guhongji\Desktop\还原设计稿"
git config --global user.name "GHVBUGB"
git add .
git commit -m "Deploy: 学生报告展示应用"
git branch -M main
git push -u origin main
```

## 🔐 获取Personal Access Token

如果提示需要认证：

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. Note: 填写 `Git Push`
4. **重要**：勾选 `repo` 权限
5. 生成并复制Token
6. 推送时作为密码使用

## ✅ 推送成功后

1. 访问：https://github.com/GHVBUGB/EA
2. 点击 **Settings** → **Pages**
3. Source 选择：**GitHub Actions** ✅
4. 等待2-3分钟自动部署完成
5. 访问：https://ghvbugb.github.io/EA/

## 📱 使用部署链接

部署完成后，在应用中：
1. 点击"接收数据"按钮
2. 在"方案2：使用部署域名"中输入：
   ```
   ghvbugb.github.io/EA
   ```
3. 粘贴JSON数据并生成链接
4. 手机可以直接打开！📱
